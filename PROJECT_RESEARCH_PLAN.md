# Graduation Project Research Plan
## Engineering Management Focus

---

## 1. PROJECT RESEARCH TITLE

**"Digital RFI & Change Order Management System: A Web-Based Solution to Mitigate Construction Project Delays"**

*Alternative: "Enhancing Construction Project Performance Through Integrated Management of RFIs and Change Orders"*

---

## 2. PROBLEM STATEMENT

### Current Issues:
1. **Request for Information (RFI) Management**
   - Unstructured communication flow between architects and contractors
   - Delayed responses leading to work stoppages
   - Lack of centralized tracking and documentation
   - Conflicts arising from miscommunication

2. **Change Order Management**
   - Disorganized approval workflows
   - Lack of transparency between stakeholders
   - Conflicts between clients, engineers, and contractors
   - Insufficient documentation and traceability
   - Unprofessional handling causing project delays

3. **Impact on Project Performance**
   - Increased project duration
   - Cost overruns
   - Resource inefficiency
   - Reduced stakeholder satisfaction
   - Professional conflicts

---

## 7. CORE FEATURES

### RFI Management Module:
1. Create RFI with detailed descriptions and attachments
2. Automatic notification system
3. Status tracking (Submitted, Under Review, Answered, Closed)
4. Comment and collaboration features
5. Historical records and analytics

### Change Order Management Module:
1. Create change orders with cost impact
2. Multi-level approval workflow
3. Impact analysis (timeline, cost, resources)
4. Version control and comparison
5. Stakeholder notifications and comments

### Common Features:
1. User authentication and role-based access control
2. Dashboard with KPIs and project metrics
3. Real-time notifications and reminders
4. Advanced search and filtering
5. Export and reporting functionality
6. Audit trail and compliance tracking
