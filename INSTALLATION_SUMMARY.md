## 🚀 Getting Started (3 Steps)

### Step 1: Setup MongoDB
- **Easy Option**: Use MongoDB Atlas (cloud)
  - Go to: https://www.mongodb.com/cloud/atlas
  - Create free account and cluster
  - Get connection string
  
- See `MONGODB_SETUP.md` for detailed instructions

### Step 2: Start Backend
```bash
cd backend
npm install
npm run dev
```

### Step 3: Start Frontend (New Terminal)
```bash
cd frontend
npm install
npm start
```

**That's it!** Your app will open at http://localhost:3000

## 📞 Support

### If Something Doesn't Work:
1. Check `QUICK_START.md` troubleshooting section
2. Verify MongoDB is running
3. Check both servers are started (port 5000 & 3000)
4. Look at browser console for errors
5. Read API_DOCUMENTATION.md for endpoint details

### Common Issues:
- **"Cannot connect to MongoDB"** → Start MongoDB or set Atlas connection
- **"Port 5000 in use"** → Change PORT in .env
- **"Frontend can't connect"** → Restart backend server
- **"Can't login"** → Register new user first
